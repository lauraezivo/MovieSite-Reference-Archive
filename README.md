# MovieSite-Reference-Archive
本仓库用于建立一个影视类网站的信息参考档案，重点记录互联网上可正常访问的电影、电视剧及综合影视站点的公开信息。所有内容均基于公开网络环境进行整理，仅用于资料存档、信息记录与结构化展示。仓库不存储任何影视内容文件，也不提供播放、跳转或引导服务，仅保留站点层面的基础信息。项目的存在目的在于长期记录与查询，便于后续进行站点信息核对与状态更新，不涉及商业推广、内容评价或版权相关立场。
